IMPORT INSTRUCTIONS FOR NEW DEVICE:
==================================

1. Prerequisites:
   - Install Java 21
   - Install Node.js 18+
   - Install MongoDB
   - Install MySQL
   - Install Maven

2. Import MongoDB:
   mongoimport --host localhost:27017 --db gradsyncdb --collection users --file mongodb_backup/gradsyncdb_users.json --jsonArray
   mongoimport --host localhost:27017 --db gradsyncdb --collection batches --file mongodb_backup/gradsyncdb_batches.json --jsonArray
   mongoimport --host localhost:27017 --db gradsyncdb --collection companies --file mongodb_backup/gradsyncdb_companies.json --jsonArray
   mongoimport --host localhost:27017 --db gradsyncdb --collection placements --file mongodb_backup/gradsyncdb_placements.json --jsonArray

3. Import MySQL:
   mysql -u root -p -e "CREATE DATABASE gradsync_db;"
   mysql -u root -p gradsync_db < mysql_backup/gradsync_db_complete.sql

4. Setup Profile Images:
   Copy profile_images/* to backend/src/main/resources/static/images/profiles/

5. Setup Configuration:
   Copy application_config/* to backend/src/main/resources/

6. Install Dependencies and Start:
   cd backend
   mvn clean install
   mvn spring-boot:run

   cd ../
   npm install
   npm run dev

7. Access Application:
   Frontend: http://localhost:3000
   Backend: http://localhost:8080
   Login: admin@piemr.edu.in / password
